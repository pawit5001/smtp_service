# SMTP Email Sender Node Server

This is the Node.js server for the SMTP Email Sender project. It serves as a proxy for requests and can also serve static files if needed.

## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd smtp-email-sender/node-server
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the server:**
   ```bash
   npm start
   ```

## API Endpoints

- Currently, this server is set up to handle proxy requests. Additional endpoints can be added as needed.

## Development

- Ensure you have Node.js installed on your machine.
- You can modify the server logic in `src/server.js` to suit your needs.
