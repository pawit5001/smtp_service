# Frontend SMTP Email Sender

This project is a frontend application built with React, TypeScript, and Tailwind CSS for sending SMTP emails. It allows users to compose their own messages or select from predefined messages for account verification and reset codes.

## Features

- User-friendly interface for composing emails.
- Option to select from predefined messages.
- Responsive design using Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm (Node Package Manager)

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd smtp-email-sender/frontend
   ```

2. Install the dependencies:
   ```
   npm install
   ```

### Running the Application

To start the development server, run:
```
npm start
```

The application will be available at `http://localhost:3000`.

### Building for Production

To create a production build, run:
```
npm run build
```

This will generate a `build` folder with the optimized application.

## Usage

- Navigate to the home page to access the email form.
- Fill in the recipient's email, subject, and body, or select a predefined message.
- Click the send button to dispatch the email.
